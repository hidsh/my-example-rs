## How to build this mdBook

### 1. Install mdBook

```shell
cd PATH/TO/FOO
cargo install mdbook
mdbook init my-rust-examples
mdbook serve --open
```

### 2. Configure mdBook
```shell
vim book.toml
```

then, add a line below.
